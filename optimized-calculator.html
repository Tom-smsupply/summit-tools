How to Use the Waste Optimizer

New to the Roof Calculator?

Please read the instructions in the Standard Calculator first to learn how to enter roof sections, measurements, and profiles.

What is the Waste Optimizer?

The Waste Optimizer helps you reduce material waste by combining panels from opposing roof sections. When you cut a panel at an angle, the offcut can be used on an opposing section instead of being wasted.

How It Works

  • Right config sections pair with Left config sections
  • Both config sections pair with other Both config sections
  • The longest angled panel from one section combines with the shortest from the opposing section

Steps

  1. Enter all your roof sections (same as the standard calculator)
  2. Set the maximum panel length your supplier can provide
  3. Add pairings between compatible opposing sections
  4. View the optimized cut list and waste savings

Section Compatibility

  • Right + Left: Angled panels can be combined
  • Both + Both: Left angles pair with right angles from opposing section
  • Both + Right + Left: If no "Both" available, a "Both" section needs two pairings
